hsptest v0.9

この度はhsptestをダウンロードしていただきありがとうございます。

hsptestとは、HSP3言語で書かれたプログラムをテストするためのフレームワークです。


■ インストール方法

hsptestは以下の手順でインストールしてください。

1. HSP3インストールフォルダ(例：C:/hsp36)を環境変数PATHに追加します
2. hsptest.zipを解凍し、install.batを実行します。
